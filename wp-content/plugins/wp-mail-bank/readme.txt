Mail Bank reconfigures the Mail Function and provides sophisticated SMTP settings to send and log emails from your WordPress site.

== Description ==

[**Mail Bank**](http://beta.tech-banker.com/products/mail-bank/) is one of the highly productive WordPress plugin.

It is one of the best plugin in the WordPress to send or receive emails.
	
The Plugin has options to choose between your custom SMTP server or the inbuilt phpmailer.

This plugin provides the easily configurable email parameters for sending emails.

It allows the view the detailed records of each and every logged emails such as Email To, Subject, Cc, Bcc, From, Debugging Output, Date/Time.

Mail Bank configures the settings to send or log emails whether successful or failure with debugging output to troubleshoot your outgoing emails.

It automatically send emails to verify setings just by a single click.

You can also configure debug mode settings to view the debugging output with complete details.

It is the primary plugin to support various Oauth 2.0 implementations such as  Gmail, Microsoft and Yahoo.

This  plugin lets you customize the email details that appear on emails sent from WordPress. With this plugin you can configure your SMTP email to use instead php mail(). 
You can customize both (from email & from name) fields with this plugin so your users are not confused. 

You can also hide your email and name by override From name and From email.

It provides two types of Encryption that are the following:

* TLS (Transport Layer Security) - It proceed to the secure connections after successful communication between Client & Server.
* SSL (Secure Sockets Layer) - It directly proceed to secure connections.

It allows users to view the complete details including Status, Delivery Report, Test Email Address, Date, MIME version and cause of unsuccessful delivery when Mail was not send due to some
problems.

It also provides Oauth authentication to send emails with more security.

This Plugin is compatible with SMTP Servers such as Gmail, Microsoft and Yahoo and many more.

== Installation ==

### Minimum requirements.
*   WordPress 2.7+
*   PHP 5.x
*   MySQL 5.x

### Performing a new installation

After downloading the ZIP file,

1. Log in to the administrator panel.
2. Go to Plugins Add > New > Upload.
3. Click "Choose file" ("Browse") and select the downloaded zip file.

*For Mac Users*
*Go to your Downloads folder and locate the folder with the plugin. Right-click on the folder and select Compress. Now you have a newly created .zip file which can be installed as described here.*

1. Click "Install Now" button.
2. Click "Activate Plugin" button for activating the plugin.

If any problem occurs, please contact us at [support@tech-banker.com](mailto:support@tech-banker.com).
